# 🏋️‍♀️ Projeto Final – Sistema de Exercícios para Academia

## 📌 Descrição
Este projeto tem como objetivo desenvolver um sistema para academia capaz de sugerir treinos personalizados de acordo com as necessidades do usuário, como objetivo físico, nível de experiência e disponibilidade de tempo.

---

## 🎯 Objetivo
Criar um software que ajude usuários a:

- Montar treinos personalizados  
- Acompanhar evolução física  
- Organizar rotinas de exercícios  
- Melhorar desempenho e saúde  

---

## ⚙️ Funcionalidades (planejadas)

- Cadastro de usuários  
- Definição de objetivos (emagrecimento, hipertrofia, etc.)  
- Sugestão automática de treinos  
- Registro de treinos realizados  
- Histórico de evolução  
- Interface simples e intuitiva  

---

## 🧠 Público-alvo

- Pessoas que frequentam academia  
- Iniciantes que precisam de orientação  
- Usuários que desejam treinos personalizados  

---

## 🛠️ Tecnologias

- Linguagem: (ex: C#)  
- Banco de Dados: (ex: SQL Server)  
- Ferramentas: Git, GitHub
